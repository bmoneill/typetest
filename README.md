# typetest: A typing speed tester

A typing speed tester. Prints a file line by line and has the
user reproduce each one. It records WPM, CPM, number of mistakes, and total
time.

## Building

```shell
cc -o typetest typetest.c
```

## Usage

```shell
./typetest file
```
